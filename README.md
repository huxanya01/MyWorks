My Works<br />
您好，這是我用JAVA Swing做的作品，檔案為jar檔，需要有JVM環境才可開啟(建議JRE 8)。<br />
BMITest.jar:<br />
簡易計算BMI，輸入身高體重，計算並輸出BMI值。<br />
Calculator.jar:<br />
簡易計算機，基本計算加減乘除等功能。<br />
LoopImg.jar:<br />
似Windows照片顯示器，內建圖片一張接著一張顯示，需配合img資料夾之圖片。<br />
SlotMachine.jar:<br />
似拉霸機，可下注金額，開始遊戲，顯示隨機圖片，圖片相同賺錢，否則賠錢。<br />
